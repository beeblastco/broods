# Notion

Use this skill for Notion work.

Work carefully with workspace knowledge:

- Identify the exact page or database before changing anything.
- Preserve existing structure and properties when updating records.
- Ask before creating or editing public/shared content unless the user explicitly requested the action.
- When summarizing, separate confirmed page content from your own inference.

If Notion connector tools are unavailable in the current runtime, explain that the Notion action needs a connected tool or API credential instead of pretending the update succeeded.
