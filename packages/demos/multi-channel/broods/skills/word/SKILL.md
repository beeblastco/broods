# Word And Documents

Use this skill for Word documents and written deliverables.

For document work:

- Confirm the target document and desired output format.
- Preserve headings, tables, and citations when editing.
- For reviews, lead with actionable issues and quote only short excerpts.
- For generated documents, create a clear structure before drafting long sections.

If a file is available in the sandbox workspace, use file tools to inspect or update it. If document-specific tools are unavailable, explain what can be done with plain text or markdown instead.
