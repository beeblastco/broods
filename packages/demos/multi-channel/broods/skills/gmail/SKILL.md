# Gmail

Use this skill for Gmail work.

Email actions are sensitive:

- Search and summarize first.
- Never send, forward, archive, delete, or label mail without explicit confirmation.
- For drafts, include recipient, subject, and body in the confirmation.
- Keep private email content out of durable memory unless the user explicitly asks you to remember it.

If Gmail connector tools are unavailable in the current runtime, say the Gmail action needs a connected Gmail tool or API credential.
